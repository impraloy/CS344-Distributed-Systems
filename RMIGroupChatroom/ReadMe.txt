The RMI Group Chatroom is a console (cmd) based java program.

Compiling Program
------------------------
To compile the program, type the following command on the console:
javac *.java

Running Program/Chatroom
------------------------
1. Run the server by typing the following command:
  
   java Server <hostname> <portnumber>

   Example:
   java Server localhost 5000

2. To run a client, type the following command:
   java Client <hostname> <portnumber>

   Example:
   java Client localhost 5000

NOTE:
- You open only one server.
- You can open multiple clients.


Exiting the ChatRoom
------------------------
Type 'exit' without the single quotes.

closing server
------------------------
Close the server by typing 'exit' (without the single quotes).
